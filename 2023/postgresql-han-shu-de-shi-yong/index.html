1 数据函数

函数名 功能 示例 结果
abs(x) 计算 x 的绝对值 abs(-17.4) 17.4
ceil(dp)/ceiling(dp) 计算大于或等于 dp 的最小整数 ceil(-42.8) -42
floor(dp) 计算小于或等于 dp 的最大整数 floor(-42.8) -43
round(dp) 四舍五入为整数 round(12.45) 12
trunc(dp) 向零取整 trunc(12.8) 12
power(a, b) 计算 a 的 b 次方 power(2, 3) 8
div(y, x) 计算 y 除以 x 的整数商 div(9,4) 2
mod(y, x) 计算 y 除以 x 的余数 mod(9,4) 1
sign(dp) 返回参数的正负号 sign(-8.4) -
random() 返回一个随机数 random() 0.07772749848663807

2 字符串函数

函数名 功能 示例 结果
concat(str, …) 用于连接字符串，并且忽略其中的 NULL 参数 concat(2, NULL, 22) 222
concat_ws(sep, str, …) 使用指定分隔符 sep 连接字符串 concat_ws(‘ and ‘, 2, NULL, 22) 2 and 22
length(string) 计算字符串包含的字符数 length(‘jose’) 4
char_length(string) 计算字符串包含的字节数 octet_length(‘jose’) 4
lower(string) 将字符串转换为小写形式 lower(‘TOM’) tom
upper(string) 将字符串转换为大写形式 upper(‘tom’) TOM
initcap(string) 将每个单词的首字母大写，其他字母小写 initcap(‘hi THOMAS’) Hi Thomas
substring(string [FROM] [for]) 用于提取从位置 FROM 开始的 for 个字符子串，位置从1 开始计算 substring(‘Thomas’ FROM 2 for 3) hom
left(str, n) 返回字符串左边的 n 个字符 left(‘abcde’, 2) ab
right(str, n) 返回字符串右边的 n 个字符 right(‘abcde’, 2) de
position(substring in string) 返回子串的位置 position(‘om’ in ‘Thomas’) 3
strpos(string, substring) 返回子串的位置 strpos(‘Thomas’, ‘om’) 3
starts_with(string, prefix) 判断 string 是否以 prefix 开头 starts_with(‘alphabet’, ‘alph’) true
replace(string, FROM, to) 将字符串 string 中的 FROM 子串替换为 to 子串 replace(‘abcdefabcdef’, ‘cd’, ‘XX’) abXXefabXXef
translate(string , FROM , to) 将字符串 string 中出现在 FROM 中的字符串替换成 to 中相应位置的字符 translate(‘12345’, ‘143’, ‘ax’) a2x5
overlay(string placing substring FROM [for]) 使用 substring 覆盖字符串 string 中从FROM 开始的 for 个字符 overlay(‘Txxxxas’ placing ‘hom’ FROM 2 for 4) Thomas
trim([leading trailing both] [characters] FROM string) 从字符串的开头（leading）、结尾（trailing）或者两端（both）删除由指定字符 characters（默认为空格）组成的最长子串 trim(both ‘xyz’ FROM ‘yxTomxx’) Tom
ltrim(string [, characters]) 与上面 trim 函数的 leading 选项相同 ltrim(‘yxTomxx’, ‘xyz’) Tomxx
rtrim(string [, characters]) 与上面 trim 函数的trailing 选项相同 rtrim(‘yxTomxx’, yxTom
lpad(string, length [, fill ]) 在 string 左侧使用 fill 中的字符（默认空格）进行填充，直到长度为 length lpad(‘hi’, 5, ‘xy’) xyxhi
rpad(string, length [, fill ]) 函数在 string 右侧使用 fill 中的字符（默认空格）进行填充，直到长度为 length rpad(‘hi’, 5, ‘xy’) hixyx
repeat(string, number) 函数将字符串 string 重复 N 次 repeat(‘Pg’, 4) PgPgPgPg
format(formatstr , formatarg) 用于对字符串格式化，类似于 C 语言中的 sprintf 函数 format(‘Hello %s, %1$s’, ‘World’) Hello World, World
md5(string) 用于返回十六进制格式的 MD5 值 md5(‘abc’) 900150983cd24fb0d6963f7d28e17f72
regexp_split_to_table(string, pattern[, flags]) 数用于拆分字符串，使用 POSIX 正则表达式作为分隔符。函数的返回类型是 text 集合 regexp_split_to_table(‘hello world’, ‘\s+’) hello
world
split_part(string, delimiter, field) 数使用 delimiter 拆分字符串，并返回指定项（从 1 开始计数） split_part(‘abc@def@ghi’, ‘@‘, 2) def
reverse(str) 用于将字符串反转 reverse(‘上海自来水’) 水来自海上

日期函数

函数名 功能 示例 结果
age(timestamp, timestamp) 计算两个时间点之间的间隔 age(timestamp ‘2020-12-31’, timestamp ‘2020-01-01’) 11 mons 30 days
age(timestamp) 计算当前日期的凌晨 12 点到该时间点之间的间隔 age(timestamp ‘2020-01-01’) 2 mons 2 days
date_part(text, timestamp)或者extract(field FROM timestamp) 用于获取日期时间中的某一部分，例如年份、月份、小时等 date_part(‘year’, timestamp ‘2020-03-03 20:38:40’) 2020
date_trunc(field, source [, time_zone ]) 用于将 timestamp、timestamp WITH time zone、date、time 或者 interval 数据截断到指定的精度 date_trunc(‘year’, timestamp ‘2020-03-03 20:38:40’) 2020-01-01 00:00:00
to_timestamp(double precision) 函数将 Unix 时间戳（自从 1970-01-01 00:00:00+00 以来的秒数）转换为 PostgreSQL 时间戳数据 to_timestamp(1583152349) 2020-03-02 20:32:29+08
