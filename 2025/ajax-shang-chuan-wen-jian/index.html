Flask + AJAX 文件上传与数据库导入完整教程

本教程将详细介绍如何使用Flask后端和AJAX前端实现文件上传并将数据导入数据库的功能。

一、系统架构概述

前端(HTML+JS) → AJAX请求 → Flask后端 → 数据库

二、环境准备

1. 安装必要库

pip install flask pandas sqlalchemy openpyxl

2. 项目结构

data_import_tool/
├── app.py                 # Flask主程序
├── upload/                # 上传文件存储目录
├── static/
│   ├── js/                # jQuery和Bootstrap JS
│   └── css/               # Bootstrap CSS
└── templates/
    └── index.html         # 前端页面

2. AJAX文件上传逻辑

$(document).ready(function(){
        // 文件选择变化时显示文件信息
        $("#formFile").change(function(){
            const file = this.files[0];
            if (file) {
                const fileInfo = `已选择: ${file.name} (${(file.size/1024/1024).toFixed(2)} MB)`;
                $("#fileInfo").html(fileInfo);
            } else {
                $("#fileInfo").html("");
            }
        });

        // 表单提交处理
        $("#uploadForm").submit(function(e){
            e.preventDefault();

            // 1. 获取文件并验证
            const file = $("#formFile")[0].files[0];
            if (!file) {
                alert("请先选择文件");
                return;
            }

            // 2. 准备FormData
            const formData = new FormData();
            formData.append("upload_file", file);
            formData.append("connection_string", $("#connectionString").val());
            formData.append("table_name", $("#tableName").val());
            formData.append("import_mode", $("#importMode").val());

            // 3. 显示加载状态
            const btn = $("#uploadBtn");
            const spinner = $("#uploadSpinner");
            const text = $("#uploadText");
            btn.prop("disabled", true);
            text.text("导入中...");
            spinner.removeClass("d-none");
            $("#progressBar").removeClass("d-none");

            // 隐藏之前的结果
            $("#resultAlert").addClass("d-none");

            // 4. 发送AJAX请求
            $.ajax({
                url: "/upload_file",
                type: "POST",
                data: formData,
                cache: false,
                processData: false,
                contentType: false,
                // 5. 进度条处理
                xhr: function(){
                    const xhr = new window.XMLHttpRequest();
                    xhr.upload.addEventListener("progress", function(evt){
                        if(evt.lengthComputable){
                            const percentComplete = Math.round((evt.loaded / evt.total) * 100);
                            $(".progress-bar").css("width", percentComplete + "%")
                                            .text(percentComplete + "%");
                        }
                    }, false);
                    return xhr;
                },
                success: function(data){
                    $("#resultAlert").removeClass("d-none alert-danger")
                                   .addClass("alert-success")
                                   .html(`<strong>导入成功！</strong> 数据已成功导入到表 ${data.table_name} 中，共导入 ${data.row_count} 行数据。`);
                },
                error: function(xhr){
                    let errorMsg = "发生未知错误";
                    try {
                        const err = JSON.parse(xhr.responseText);
                        errorMsg = err.error || err.message || "导入失败";
                    } catch(e) {
                        errorMsg = xhr.responseText || "导入失败";
                    }

                    $("#resultAlert").removeClass("d-none alert-success")
                                   .addClass("alert-danger")
                                   .html(`<strong>导入失败！</strong> ${errorMsg}`);
                },
                complete: function(){
                    btn.prop("disabled", false);
                    text.text("导入数据");
                    spinner.addClass("d-none");
                }
            });
        });
    });

四、后端实现详解

1. Flask应用配置

import os
import json
import pandas as pd
from sqlalchemy import create_engine, exc
from flask import Flask, render_template, request, Response
from datetime import datetime
from werkzeug.utils import secure_filename

app = Flask(__name__)
app.config['UPLOAD_FOLDER'] = 'upload'
app.config['MAX_CONTENT_LENGTH'] = 50 * 1024 * 1024  # 50MB文件大小限制
os.makedirs(app.config['UPLOAD_FOLDER'], exist_ok=True)

# 允许的文件扩展名
ALLOWED_EXTENSIONS = {'xlsx', 'xls', 'csv'}

2. 文件上传路由

@app.route('/upload_file', methods=['POST'])
def upload_file():
    try:
        file = request.files['upload_file']
        # 1. 验证文件
        if not allowed_file(file.filename):
            return Response(json.dumps({'error': '不支持的文件类型'}),
                            status=400, mimetype='application/json')

        # 2. 获取表单数据
        connection_str = request.form.get('connection_string')
        table_name = request.form.get('table_name')
        import_mode = request.form.get('import_mode', 'replace')

        # 3. 保存文件
        filename = generate_timestamp_filename(file.filename)
        file_path = os.path.join(app.config['UPLOAD_FOLDER'], filename)
        file.save(file_path)

        # 4. 导入数据库
        row_count = import_to_database(
            file_path=file_path,
            connection_str=connection_str,
            table_name=table_name,
            import_mode=import_mode
        )

        # 5. 返回成功响应
        return Response(json.dumps({
            'success': True,
            'message': '数据导入成功',
            'table_name': table_name,
            'row_count': row_count,
            'file_path': file_path
        }), mimetype='application/json')

    except Exception as e:
        return Response(json.dumps({
            'error': str(e),
            'message': '数据导入失败'
        }), status=500, mimetype='application/json')

3. 核心功能函数

def allowed_file(filename):
    return '.' in filename and \
           filename.rsplit('.', 1)[1].lower() in ALLOWED_EXTENSIONS

def generate_timestamp_filename(original_filename):
    """生成基于时间戳的文件名"""
    filename = secure_filename(original_filename)
    name, ext = os.path.splitext(filename)
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S_%f")[:-3]
    return f"{name}_{timestamp}{ext}"

def read_data_file(file_path):
    """根据文件扩展名读取数据文件"""
    ext = os.path.splitext(file_path)[1].lower()

    if ext == '.csv':
        return pd.read_csv(file_path)
    else:  # .xlsx or .xls
        return pd.read_excel(file_path)

def import_to_database(file_path, connection_str, table_name, import_mode):
    """将数据导入数据库"""
    try:
        # 读取文件
        df = read_data_file(file_path)

        # 创建数据库引擎
        engine = create_engine(f"{connection_str}")

        # 导入数据
        row_count = len(df)
        df.to_sql(
            name=table_name,
            con=engine,
            index=False,
            if_exists=import_mode,
            chunksize=1000
        )

        return row_count
    except exc.SQLAlchemyError as e:
        raise Exception(f"数据库错误: {str(e)}")
    except pd.errors.EmptyDataError:
        raise Exception("文件为空或格式不正确")
    except Exception as e:
        raise Exception(f"数据处理错误: {str(e)}")

五、部署与运行

启动Flask应用：

python app.py

访问应用：

http://localhost:5000

使用流程：

  • 输入数据库连接字符串
  • 设置目标表名和导入模式
  • 选择Excel/CSV文件
  • 点击”导入数据”
  • 查看导入结果

六、常见问题解决

  1. 文件上传失败：
    • 检查上传文件夹权限
    • 确认文件大小不超过限制
    • 查看浏览器控制台和Flask日志
  2. 数据库连接问题：
    • 验证连接字符串格式
    • 检查数据库服务是否运行
    • 确认网络连接
  3. 数据导入错误：
    • 检查Excel/CSV文件格式
    • 确认表结构与数据匹配
    • 查看是否有特殊字符需要处理
